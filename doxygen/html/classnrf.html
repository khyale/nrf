Descrição detalhada

Classe nrf.

Esta classe possui métodos de configuração controle para o chip nordic nRF24L01+. Foi implementada e testada no Arduino Uno.

void nrf::enable_rx_pipe ( nrf_address_t  pipe,
bool  auto_ack 
)

Habilita pipe.

Utilize esta função para habilitar um pipe de recepção. O chip possui 6 pipes que podem ser utilizados para receber pacotes de até 6 dispositivos distintos, numa arquitetura conhecida como Multiceiver (consulte datasheet).

Parâmetros
[in]pipePipe de recepção
[in]auto_ackHabilita ou não a função 'auto acknoledgment'. Se auto_ack for 'true', após o pacote ser recebido, o PRX enviará um pacote ACK de volta para o PTX.
Aviso
Esta função configura os bits dos registradores EN_RXADDR e EN_AA de acordo com o 'pipe' informado.

Definido na linha 926 do ficheiro nrf.cpp.

void nrf::set_mode ( nrf_operation_mode_t  mode)

Configura o modo de operação do dispositivo.

Parâmetros
[in]modeModo de operação.

Os modos de operação recomendados são:

  • NRF_POWER_DOWN. Este modo é o estado inicial e de menor consumo.
  • NRF_STANDBY. Este modo é alcançado fazendo-se o bit PWR_UP=1 e CE=0. O consumo é intermediário pois o oscilador já estará ligado.
  • NRF_RX_MODE. Modo de recepção. Neste estado é possível receber pacotes.
  • NRF_TX_MODE. TX mode Modo de transmissão. Neste modo o dispositivo monta os pacotes a partir do FIFO de TX e os envia. Quando o FIFO está vazio, o dispositivo automaticamente vai para um estado de menor consumo.
